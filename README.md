## 搬瓦工注册地址怎么填：逐字段说明

进入注册页面，你会看到这些字段。按下面的方式处理就行：

### First Name / Last Name（名字 / 姓氏）

填你的真实姓名拼音就行，或者用英文名。不要写成 "aaa" 或 "dfskjf" 这种一眼假的乱码——搬瓦工是老外公司，看到这种名字真的会怀疑。

比如你姓张名伟，写 **Zhang Wei** 完全没问题。

### Email Address（邮箱）

必须填真实可用的邮箱。购买成功后，VPS 的 IP 地址、root 密码、SSH 端口都会发到这个邮箱。国内的 QQ 邮箱、163 邮箱都可以正常收到。

### Password / Confirm Password（密码）

设一个有大小写+数字的密码，没别的要求。

### Country（国家）

**必须选 China（中国）**，并且注册时不要挂代理，保证 IP 和国家匹配。这是最关键的一个字段。

### State / Region（省份）

填你所在省份的拼音或英文名。比如广东是 Guangdong，北京是 Beijing，不用太精确，但要和城市对得上。

### City（城市）

填你当前所在城市的拼音。不需要精确到街道，和省份以及 IP 地址大致匹配即可。比如你在上海，就填 Shanghai。

我自己用的是北京的 IP，City 填的北京，State 填的北京，一次过没触发任何风控。

### Address 1 / Address 2（详细地址）

这两个填什么都行，随便编一个看起来正常的地址就够了。可以用你家附近的路名拼音，比如 "123 Zhongshan Road"。搬瓦工不会用这个做验证。

### Zip Code（邮编）

随便搜一下你所在城市的邮编填进去。比如北京朝阳区是 100020，上海是 200000，不需要精确到小区。

### Phone Number（电话）

这个字段对欺诈检测影响不大，区号填对就行。国内手机号直接填：**+86 13xxxxxxxxx**（格式带国际区号）。实在不想填真实的，随便写个看起来正常的也不会出问题。

IP 在美国，但国家选了中国——必然触发 Fraud，这种最常见。

搬瓦工官网首页的注册流程在国内有个 bug——验证码是谷歌的，国内用户打不开，注册就卡死了。

正确做法是**先选好套餐加购物车，结账时再填注册信息**。这个流程绕开了首页的验证码问题，是国内用户常用的正确姿势。
